# 🤖 Week 1: Rule-Based AI Chatbot

Welcome to the repository for **Week 1** of my Artificial Intelligence Internship. This week's practical project focused on understanding foundational AI logic by building a **Rule-Based AI Chatbot** using Python.

---

## 📌 Project Overview
The goal of this project is to develop a text-based conversational agent that responds intelligently to predefined user inputs. Unlike machine learning models that learn from data, this chatbot operates on a deterministic network of conditional rules, simulating an interactive digital assistant.

### 🎯 Key Requirements
*   **Greeting & Exit Controls**: Successfully handles standard conversation initiations (e.g., hello, hi) and safe closure commands (e.g., bye, exit).
*   **Decision Logic**: Built using sequential `if-else` and `elif` conditional control structures to map input keywords to relevant responses.
*   **Continuous Execution**: Runs inside an infinite state loop (`while True`) ensuring continuous dialogue until explicitly terminated by the user.

---

## 🛠️ Tech Stack & Key Skills
*   **Language**: Python 3.x
*   **Core Logic**: Control flow, conditional decision-making frameworks, and string manipulation (case folding/lowercasing for robust matching).
